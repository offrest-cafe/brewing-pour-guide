01

커피 맛은 순서대로 나온니다

커피에 문을 부으메 성분이 한꺼번에 나오지 않습니다.
작은 분자가 먼저, 큰 분자가 나중에 녹아 나옵니다.

순섞성분맛쉬운 비유
초반 유기산 (작은 분자) 산미 — 신맛 레몬즙의 첫 신맛
중반 당류 / 칰러멜 단맛 — 스위 스팟 쿤키 굴는 고소한 향
후반 타닌 / 카페인 (큰 분자) 쓴맛 — 떫은맛 티백 너무 오래 우린 맛

기억법: 산(신맛) → 단(단맛) → 쓴(쓴맛)

추출이 길어질수록 쓴맛이 늘어납니다. 차 티백과 같은 원리입니다.

02

기준 레시피 분해

오프레스트 기본 레시피를 구간별로 분해하면 이렇습니다.

구간물의 양누적추출되는 맛역할
1차 (뜸) 50g 0 → 50g — CO₂ 방출 + 물 침투 준비
2차 70g 50 → 120g 산미 + 단맛 맛의 방향을 결정
3차 120g 120 → 240g 단맛 + 바디 농도와 무게감을 채움

2차 푸어 = 맛의 방향 (신맛/단맛 밸런스)

3차 푸어 = 농도 (진하게 or 가볍게)

03

상황별 조절법

"단맛을 더 느끼고 싶어요"
건드리는 곳  2차 푸어

2차를 2번으로 나눠서 천천히 붓기

기존: 50g → 120g (한번에 70g)

변경: 50g → 80g (30g) → 130g (50g) — 각각 천천히

원리: 단맛 성분(당류)은 추출 중반에 나옵니다. 천천히 부으면 단맛이 충분히 녹아나올 시간이 생깁니다.

3차를 130g → 240g(110g)으로 줄여서 총량 240g 유지할 것
"신맛이 너무 강해요"
건드리는 곳  1차 뜸 + 물 온도

A) 뜸 물량 늘리기: 50g → 55~60g

CO₂가 더 빠지고 초반 산미 성분이 뜸 문에 녹아 빠져나감

B) 물 온도 2도 낭추기: 93도 → 91도

온도가 낮으면 산미 성분이 덜 추출됨

원리: 유기산은 분자가 작아 가장 먼저 녹아나옵니다. 뜸에서 미리 빼주메 이후 푸어에서 산미가 줄어듭니다.

뜸 물 70g 이상은 금지 — 맛이 물어집니다
"쓴맛이 강해요 / 떫어요"
건드리는 곳  3차 푸어

A) 3차를 빠르게 한번에 확 붓기

물이 빨리 빠지메 쓴맛 성분이 덜 추출됨

B) 총량 240g 대신 230g에서 컷

마지막 10g의 쓴맛/떫은맛을 아예 제거

C) 분쇄도를 한 칸 굵게

물이 빨리 빠져 전체 추출 시간이 줄어듦

원리: 쓴맛 성분(타닌, 카페인)은 분자가 커서 항상 마지막에 나옵니다.

문이 거의 다 빠졌을 때 드리퍼 제거. 너무 일찍 빼메 묽어짐
"묵직하고 진하게 마시고 싶어요"
건드리는 곳  3차 푸어 + 분쇄도

3차를 천천히 느리게 붓기

젝촉 시간이 늘어나 더 많은 성분이 녹아나옴

분쇄도 한 칸 가늘게

물이 천천히 빠져 젝촉 시간 추가 증가

추출 시간 4분 넘기면 쓴맛이 급즥합니다
"깨끗하고 가볍게 마시고 싶어요"
건드리는 곳  3차 푸어 + 분쇄도

3차를 한번에 확 붓고 빠르게

무거운 성분이 덜 추출됨

분쇄도 한 칸 굵게

물이 빠르게 빠져 전체 추출 시간 단축

손니 요청건드리는 곳방법주의
단맛 늘려주세요 2차 푸어 2차를 2번 나눠 천천히 총량 240g 유지

쓴맛 줄여주세요 3차 푸어 3차 빠르게 / 230g 컷 4분 넘기메 위험
묵직하게요 3차 + 분쇄도 3차 천천히 / 분쇄 가늘게 4분 넘지 말 것
가볽하게요 3차 + 분쇄도 3차 빠르게 / 분쇄 굵게 총량 250g까지 OK

01

핫 vs 아이스 — 무엇이 다른가

같은 원두라도 온도와 추출 방식에 따라 맛이 완전히 달라집니다.
아이스는 진하게 추출한 뒤 급냉한여 깔끔한 맛을 만듭니다.

항목핫 브루잉아이스 브루잉
원두16g22g
총 문240g (3회)200g (4회)
문 온도93°C92°C
푸어링50g → 70g → 120g50g × 4회 (30초 간격)
총 시간~3분~2분 10초
서빙바로 서빙스월링 후 얼음잔에 서빙

핵심: 서버에 얼음을 넣지 않습니다. 추출 후 얼음잔에 부어 급냉시키는 방식입니다.

02

기본 레시피 (4회 균등 푸어링)

22g 원두 / 200g 문(92°C) / 4회 × 50g / 30초 간격

구간물의 양누적타이밍역할
1차 (뜸) 50g 0 → 50g 0:00 CO₂ 방출 + 원두 침투
2차 50g 50 → 100g 0:30 산미 + 단맛 추출
3차 50g 100 → 150g 1:00 바디감 채우기
4차 50g 150 → 200g 1:30 마무리
드리퍼 제거 — ~2:10 원두 표면 드러나면 제거

규칙: 30초마다 정확히 50g씩. 타이머 필수.

2분 10초쯤 원두 베드가 드러나면 드리퍼를 제거합니다.

03

왜 이렇게 내리는가

원두 22g: 핫(16g)보다 많은 이유 — 얼음에 희석되므로 진하게 추출해야 밸런스가 맞습니다.

물 200g (92°C): 총량을 줄이고 온도를 1도 낮춰서 과추출 없이 농축된 맛을 만듭니다.

4회 균등 푸어링: 핫과 달리 매회 같은 양(50g)을 부어 균일한 추출을 만듭니다. 변수를 최소화합니다.

스월링 서빙: 추출 완료 후 서버를 가볍게 스월링(돌려주기)하여 농도를 균일하게 만든 뒤, 얼음이 가득 찬 서빙잔에 부어줍니다.

04

바 체크리스트 (STEP 1–8)

프린트해서 바 근처에 붙여두세요

STEP 1
원두 22g 계량
핫(16g)보다 6g 더 — 아이스 전용 세팅
STEP 2
중간 굵기 분쇄
핫과 동일한 분쇄도 유지
STEP 3
물 92°C 준비
핫(93°C)보다 1도 낮게 — 과추출 방진
STEP 4
드리퍼 + 필터 린싱
서버에 얼음 넣지 않음 — 빈 서버 위에 드리퍼 세팅
STEP 5
1차 50g (뜸들이기)
타이머 시작. 원두 전체 적시기
STEP 6
2차~4차 각 50g (30초 간격)
0:30 → 100g / 1:00 → 150g / 1:30 → 200g
STEP 7
~2:10 드리퍼 제거
원두 표면이 드러나면 즉시 제거
STEP 8
스월링 후 얼음잔에 서빙
서버를 가볍게 돌려 농도 균일화 → 얼음 가득 찬 잔에 부어 급냉
05

절대 하지 말 것

서버에 얼음을 넣고 추출하지 마세요. 추출 후 얼음잔에 부어야 맛이 깨끗합니다.
총 물의 양(200g)을 함부로 바꾸지 마세요. 비율이 깨집니다.
30초 간격을 무시하지 마세요. 타이밍이 맛의 핵심입니다.
추출 시간 2분 30초를 넘기지 마세요. 쓴맛이 급증합니다.
